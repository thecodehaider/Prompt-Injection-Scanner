# Security Policy

## Supported Versions

The hosted Prompt Injection Scanner service at [quantizelab.dev/tools/prompt-scanner](https://quantizelab.dev/tools/prompt-scanner) is continuously maintained and updated.

## Reporting a Vulnerability

We take security seriously. If you discover a vulnerability in our scanning service, API, or web application:

**Please do NOT open a public GitHub issue for security vulnerabilities.**

Instead, report privately via:

📧 **Email**: [thecodehaider@gmail.com](mailto:thecodehaider@gmail.com)  
**Subject**: `[SECURITY] Vulnerability Report — Prompt Injection Scanner`

### What to Include

- A description of the vulnerability
- Steps to reproduce
- Potential impact
- Any suggested mitigations (optional)

### What to Expect

- Acknowledgment within **48 hours**
- Status update within **7 days**
- Credit in our security acknowledgments (if desired)

### Scope

In-scope:
- The REST API at `www.quantizelab.dev/api/v1/scan`
- The web application at `www.quantizelab.dev`
- Authentication and authorization issues
- Data exposure or leakage

Out of scope:
- Denial of service attacks
- Social engineering
- Issues requiring physical access

---

Thank you for helping keep the LLM security ecosystem safe.
